# Dane badawcze

## Zawartość

Katalog zawiera dane udostępnione publicznie w związku z realizacją projektu GoGeo.

### `01_dane_surowe`

Pliki pochodzące bezpośrednio z pomiarów lub rejestracji.

### `02_dane_przetworzone`

Pliki przygotowane na podstawie danych surowych, np. zestawienia, statystyki i wyniki obliczeń.

## Wymagany opis każdego zbioru

Każdy zestaw danych powinien zawierać:

- nazwę badania,
- datę,
- opis warunków,
- opis formatu,
- opis kolumn,
- jednostki,
- sposób przetwarzania,
- powiązanie z raportem lub wynikiem badania.

## Zalecane formaty

CSV, TXT, JSON, PDF.

Formaty zamknięte powinny być stosowane tylko dodatkowo.
